# 🚗 Car Rental System

A simple Python-based **Command-Line Car Rental Management System** that lets users view available cars, rent a car, return a rented car, and manage rental records.  

This project is a beginner-friendly Python application focused on object-oriented design, file handling (or simple data storage), and basic menu-driven interactivity.

---

## 🧠 Features

✔️ View all cars and their availability  
✔️ Rent a car  
✔️ Return a rented car  
✔️ Maintain rental history  
✔️ Simple CLI menu interface  
✔️ Organized using classes and functions
